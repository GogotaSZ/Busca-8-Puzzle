# 8-Puzzle Busca IA

8-Puzzle Busca IA é um projeto acadêmico desenvolvido para a disciplina de Inteligência Artificial. O objetivo do projeto é implementar algoritmos de busca para resolver o problema do 8-Puzzle, permitindo encontrar uma sequência de movimentos entre um estado inicial e o estado objetivo.
